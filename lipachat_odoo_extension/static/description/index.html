LipaChat WhatsApp Gateway

Send WhatsApp messages through LipaChat Gateway

This module integrates Odoo with LipaChat WhatsApp Gateway API to:

* Send text messages

* Send media messages (images, videos, documents)

* Create and manage message templates

* Track message delivery status

Features

* Easy API key configuration

* Contact integration with WhatsApp numbers

* Message history tracking

* Template management

Setting Up

Step 1: Create an account at https://app.lipachat.com/auth/signup

Step 2: Get your API key at https://app.lipachat.com/app/settings

Step 3: Enter your API key in odoo configuration and test the connection

Step 4: Configure your WhatsApp Business number here: https://app.lipachat.com/app/golive OR you can use our default test number

Step 5: Start sending WhatsApp messages/templates from Odoo
